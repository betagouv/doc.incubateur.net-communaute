# Support tech

Pour le support technique lié aux [services fournis par beta.gouv.fr](../../../les-outils-de-la-communaute/les-outils-de-la-communaute.md), fais ta demande directement via l'[espace-membre](https://espace-membre.beta.gouv.fr/services/ops). Pour toute question, utilise le canal [Demandes-OPS sur Tchap](https://tchap.gouv.fr/#/room/!VxFWdbcSlumKPvpVRP:agent.dinum.tchap.gouv.fr?via=agent.dinum.tchap.gouv.fr).

Pour les questions techniques d'ordre général, tu peux solliciter la communauté via [des canaux dédiés](../../../les-outils-de-la-communaute/mattermost/canaux-recommandes.md), par exemple [`Domaine-dev` sur Tchap](https://tchap.gouv.fr/#/room/!RZYCMTpOqgMrCofbMs:agent.dinum.tchap.gouv.fr?via=agent.dinum.tchap.gouv.fr&via=agent.social.tchap.gouv.fr&via=agent.finances.tchap.gouv.fr).

Vous pouvez également solliciter [le référent(e) tech](to-do-liens-avec-les-referents-techs.md) de votre structure.
